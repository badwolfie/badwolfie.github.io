About me

I'm a Computer Systems Engineering undergraduate and software developer, I started programming at age 17 almost by accident and I just fell in love with it. Recently, I decided to start some personal open source software development projects in my spare time in order to learn as much as I can from them and also get to interact and collaborate with other software developers from around the world.

As for professional experience, I'm currently working as a software developer for Ingenico Group. My previous experience include collaborating, also, as a software developer in some projects for the Centro de Investigación en Computación (CIC-IPN), from my classes and projects back when I was studing at the Escuela Superior de Cómputo (ESCOM-IPN) and from the personal projects I've been working on.

I mainly use (and love) C to write my code but sometimes I also use C++, Java, Vala or other languages, depending on what I want to do.

My current OS setup is Arch Linux + GNOME.
